# Weekly E-commerce Overview (BigQuery SQL)

This project builds a weekly dataset to analyze:
- sessions (sessions_cnt)
- purchasing sessions (orders_cnt)
- revenue

## Dimensions

week_start, channel, device

## Output

1 row per week_start, channel, and device.

## Files

- `query.sql` — BigQuery SQL query
- `dashboard.png` — Looker Studio dashboard screenshot
